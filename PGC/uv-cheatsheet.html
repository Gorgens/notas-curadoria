⬇ Install uv
macOS / Linux
curl -LsSf https://astral.sh/uv/install.sh | sh
Windows (PowerShell)
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
via pip
pip install uv
via Homebrew
brew install uv
update uv itself
uv self update
🐍 Python Management
list available versions
uv python list
list installed only
uv python list --only-installed
install version
uv python install 3.12
pin project to version
uv python pin 3.12
find python path
uv python find
📁 Projects
create new project
uv init myproj
init in current dir
uv init .
new project + pin python
uv init myproj && uv python pin 3.12
sync / install deps
uv sync
lock dependencies
uv lock
📦 Package Management
add package(s)
uv add django requests
add with version constraint
uv add "pandas>=2.3"
add dev dependency
uv add --dev pytest ruff
add from git
uv add "git+https://github.com/user/repo"
add optional extras
uv add "httpx[http2]"
remove package
uv remove django
upgrade package
uv add --upgrade django
upgrade all
uv sync --upgrade
list installed
uv pip list
show dependency tree
uv tree
▶ Run
run script (auto-venv)
uv run main.py
run module
uv run -m pytest
run with extra dep (temp)
uv run --with rich main.py
run w/ specific python
uv run --python 3.11 main.py
interactive python shell
uv run python
🛠 Tools (uvx)
run tool without install
uvx ruff check .
equivalent (explicit)
uv tool run ruff check .
run specific version
uvx "ruff==0.4.0" check .
run with extras
uvx --from "black[jupyter]" black .
install tool globally
uv tool install ruff
list installed tools
uv tool list
upgrade tool
uv tool upgrade ruff
uninstall tool
uv tool uninstall ruff

🔄 Typical Project Workflow
1
uv init myproj
Create project structure with pyproject.toml
2
uv python pin 3.12
Pin Python version (.python-version)
3
uv add django
Add deps → auto-creates venv + uv.lock
4
uv add --dev pytest ruff
Add dev dependencies separately
5
uv run main.py
Runs in venv, auto-installs if missing
6
uv sync
Sync others' env from uv.lock on clone
🚩 Common Flags & Options
Flag Description Example
--dev Add as dev dependency uv add --dev pytest
--upgrade / -U Force upgrade packages uv sync --upgrade
--python / -p Specify Python version uv run --python 3.11 script.py
--with Add temporary dep for run uv run --with rich main.py
--no-sync Skip syncing venv uv run --no-sync main.py
--frozen Use lockfile as-is, no update uv sync --frozen
--all-extras Install all optional extras uv sync --all-extras
--quiet / -q Suppress output uv add -q requests
